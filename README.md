# Website

## What I Built ##

This project included the creation from design to hosting of a static website as part of my Web Programming I course. The project focused on building, responsive, clear, and accessible web pages using modern HTML and CSS. 

## Skills Devoleped in this Process ##
- Structuring content with clean, semantic HTML
- Styling with organized, reusable CSS
- Designing with intentional typography, colour, spacing, and overall layout
- Creating simple, intuitive navigation
- Editing and optimizing digital images
- Embedding multimedia content (video, sound, photography)
- Applying basic CSS libraries (Bootstrap) to support faster, consistent styling
- Implementing mobile first design following industry standards

## Key Takeaways ##

This project gave me practical experience with the core tools and priciples behind static web development. I learned how to follow a consistent development to get satisfactory predictable results. This translates to an ability to understand how each of the above skills of web development work together to create a clear user experience. 

In doing a project of this caliber I learned neccessary time management and workflow planning skills as well. Breaking tasks into manageable steps, writing organized, maintainable code, and working with a deadline prepared me for a professional environment.
